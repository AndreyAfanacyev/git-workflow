# Отчет по работе с Git и GitHub

## 1. Создание репозитория
Репозиторий создан на GitHub с названием `git-workflow`.

## 2. Первоначальная настройка
- Репозиторий клонирован на локальный компьютер.
- Создан файл `README.md` и выполнена первая операция commit.

## 3. Работа с ветками
- Создана ветка `feature-branch`.
- Внесены изменения в файл `README.md` в ветке `feature-branch`.
- Выполнен commit с описанием изменений.

## 4. Слияние веток
- Ветка `feature-branch` успешно слита с основной веткой `main`.

## 5. Итог
- Все изменения отправлены в удаленный репозиторий на GitHub.
